# NTU COOL 檔案預覽小幫手 (NTU COOL File Previewer)

專為 NTU COOL（基於 Canvas LMS）設計的瀏覽器擴充功能。本套件讓學生與使用者不需下載檔案，即可直接在瀏覽器內快速預覽多種格式的附件內容，大幅提升學習與批改作業的效率。

## 特色功能

* **雙區域無縫支援**：
  * **作業繳交區**：在已繳交的檔案旁，針對支持的副檔名檔案自動生成「預覽」按鈕，點擊後以彈出式視窗顯示，並支援自由拖曳與縮放視窗大小。
  * **課程內容區**：允許使用者直接在內文區域自動載入並展開預覽畫面。
* **廣泛的格式支援**：
  * **程式碼與純文字**：py, c, cpp, js, html, css, json, md, v, sv, txt, csv, ipynb 等（支援語法高亮與一鍵複製）。
  * **試算表**：xls, xlsx（支援多工作表切換預覽）。
  * **簡報**：ppt, pptx。
  * **壓縮檔**：zip, tgz, gz。
  * **多媒體**：常見的圖片（png, jpg, webp 等）、影片（mp4, webm）與音檔（mp3, wav）。
* **沉浸式閱讀**：提供「全螢幕」功能，並內建畫面的放大/縮小工具，方便檢視圖表或小字體程式碼。

## 限制與已知問題

由於所有檔案解析皆在「前端瀏覽器」完成，本套件具有以下限制，使用時請留意：

1. **PowerPoint 簡報 (PPT/PPTX)**：採用前端轉譯渲染，複雜的排版、自訂字體、動畫與特殊圖形可能會跑版或無法完美呈現。僅適合用來快速瀏覽簡報文字與基本架構。
2. **壓縮檔 (ZIP/TGZ/GZ)**：提供樹狀結構預覽與查看內部檔案。僅支援部分檔案的預覽。
3. **Excel 試算表 (XLS/XLSX)**：僅提供純表格與文字資料的預覽，無法顯示圖表 (Charts)、巨集 (Macros) 或過於複雜的儲存格樣式。
4. **大檔案效能**：由於依賴使用者的電腦運算能力，開啟過大的簡報檔、Excel 或含有數萬個檔案的壓縮包時，可能會有短暫卡頓或記憶體消耗較大的情況。
5. **注入式視窗差異**：由本插件注入的預覽視窗，雖已儘可能模仿原生的官方視窗，但仍可見介面及操作邏輯上的差異。

## 安裝教程
本套件支援所有 Chromium 核心的瀏覽器（如 Google Chrome、Microsoft Edge、Brave 等），可直接透過「開發人員模式」拖曳安裝，無需依賴第三方腳本管理器：

1. 將本腳本壓縮檔解壓，並確保manifest.json在資料夾的第一級結構中。
2. 開啟您的瀏覽器，在網址列輸入並前往擴充功能管理頁面：
   * Chrome 瀏覽器：`chrome://extensions/`
   * Edge 瀏覽器：`edge://extensions/`
3. 開啟頁面右上角（或左下角）的「**開發人員模式**」(Developer mode) 開關。
4. 將剛剛儲存好的資料夾，直接「拖曳」到擴充功能頁面的視窗中。
5. 瀏覽器會跳出確認安裝的提示框，點擊「新增擴充功能」即可完成安裝並開始使用。
6. 打開NTU COOL網站即刻生效插件。您可點選擴充圖標圖示來編輯支援的副檔名。
7. 在支援的副檔名列表中新增或移除zip/xls/xlsx可屏蔽或取消屏蔽原生的預覽系統。

## 使用的開源套件

本專案的實現依賴於以下開源社群套件
* Highlight.js (用於程式碼語法高亮): https://highlightjs.org/
* JSZip (用於解析 ZIP 壓縮檔): https://stuk.github.io/jszip/
* Pako (用於解壓縮 TGZ/GZ 檔案): https://github.com/nodeca/pako
* SheetJS (用於讀取 Excel): https://sheetjs.com/
* PPTXjs (用於渲染 PowerPoint): https://pptxjs.com/
