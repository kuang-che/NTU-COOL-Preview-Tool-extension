NTU COOL Preview Tool 是一個完全在使用者本地瀏覽器運作的開源擴充功能。本套件不會收集、儲存、追蹤或傳輸使用者的任何個人資料、登入憑證或瀏覽紀錄。所有的檔案解析與預覽皆在本地端完成。使用者自訂的副檔名設定僅透過 Chrome 內建的 storage API 儲存於本地設備中。

NTU COOL Preview Tool is an open-source extension that operates entirely within the user's local browser. This extension does not collect, store, track, or transmit any personal data, login credentials, or browsing history. All file parsing and previews are processed locally. User-customized file extension settings are stored exclusively on the local device using Chrome's built-in storage API.
